# Biophysics_Simulations

Computational and experimental simulations in biophysics. Includes molecular dynamics,
protein folding, biomolecular interactions, and data analysis using Python, MATLAB, and GROMACS.

## Directory Structure
- `simulations/`: Molecular dynamics and biophysics simulation files
- `scripts/`: Scripts for analyzing simulation data
- `data/`: Raw and processed datasets
- `notebooks/`: Jupyter notebooks for experiments and visualization
- `plots/`: Graphical representations of results
- `docs/`: References, theory, and usage guides

## Simulation Examples
- Protein folding/unfolding
- Ligand-receptor interactions
- Diffusion simulations
- Molecular dynamics using GROMACS or LAMMPS

## Analysis
- RMSD/RMSF of proteins
- Energy calculations
- Structural visualizations

- **Molecular Dynamics**: Share MD workflows and parameter files
- **ML for Physics and Chemistry**: Apply machine learning to predict protein structure or properties
